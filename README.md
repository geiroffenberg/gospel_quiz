# gospel_quiz

New Testament verse reference quiz built with Flutter.

## Data Scope

- Quiz set: top 777 New Testament verses in [assets/top_777_nt.json](assets/top_777_nt.json)
- Chapter/verse dropdown structure: [assets/bible_structure_nt.json](assets/bible_structure_nt.json)

## Run

1. Install dependencies:
	flutter pub get
2. Launch app:
	flutter run
